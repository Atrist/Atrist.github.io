# React VS Vue

1. 如果你喜欢用(或希望能够用)模板搭建应用,请使用Vue
2. 如果你喜欢简单和"能用就行"的东西,请使用Vue
3. 如果你的应用需要尽可能的小和快,请使用Vue
4. 如果你计划构建一个大型的应用程序,请使用React
5. 如果你想要一个更大的生态圈,请使用React
